What is Alt Text?

Alt text (or alternative text) is a feature employed by website designers to help describe the visual content of an image. Designers add a brief piece of text that explains an image which allows people with visual impairments to still understand the purpose of an image while not being able to see it. The text is read to them by software's called "screen readers." Your alt text should be kept brief, ideally less than 150 characters, and should avoid restating already available information. For example, screen readers can identify the presence of an image, and thus starting alt text with "An image of..." is redundant. It is incredibly important that you add it to your digital content, as it makes the website easier to understand by all.

Alt text is important because it allows those with visual impairments to experience content of a web page that would otherwise be unavailable to them. With this in mind, it is especially important that you write alt text correctly so that their experience is a complete one. Use the following tips to help you in the process!

The Rules of Alt Text1

  1. Be specific and efficient

    1. Give the information the image provides.

  2. Don't start with "image/picture of..." as that is redundant

    1. The user will already be aware it is in image as the program they use will tell them so

  3. Include text that is part of the image

  4. Avoid repetition

    1. If an image already has a descriptive caption, you don't need to add alt text as that would provide no value to the user.

  5. Don't add alt text to decorative images

    1. Background images or dividers don't need alt text as they don't add helpful information, and may just clutter the screen reader. In this case, you leave the alt text blank (null alt text) like the following alt=""/>

Good vs. Bad Alt Text

If we look at the example image above, the user is attempting to write alt text to describe a picture of pizza. At a glance it seems to do its job, but if you re-read it then it's clear this is bad alt text. There is actually one slice of pizza missing, instead of one slice on the plate. It does not describe the image properly, giving a misleading interpretation of it. When writing good alt text, we want to ensure we describe the image accurately, while being concise. For example, "A black and white pizza on a tray with one slice missing." This provides a relatively short, but informative explanation of the image that is most importantly true. Other common mistakes when it comes to writing alt text is keeping things too short, or leaving out details. For example, "Pizza" or "Pizza on a plate," while accurate, lack the extra context provided in the image, and are bad pieces of alt text as a result.
