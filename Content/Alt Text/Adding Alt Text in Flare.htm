Adding Alt Text in Flare

Adding alt-text to images is key to improve the accessibility of your website. It allows those with visual impairments to understand the context of your images, and thus knowing how to add it is important. There are multiple ways to add alt text, depending on whether an image has been inserted, or the editor you prefer.

Adding Alt-Text to an Inserted Image (XML Editor)

  1. Right click an image.

  2. Click Image Properties.

The following pop up will appear

3. Type your alt text into the Alternate Text box under Options.

4. Click Ok.

Adding Alt-Text to a New Image (XML Editor)

  1. Click the Insert tab.

  2. Click Image.

  3. Select your image using the Select File menu.

  4. Type your alt-text into the Alternate Text box under Options.

  5. Click Ok.

Adding Alt-Text Using the Text Editor

  1. Click the Text Editor tab at the bottom of the editor window.

  2. Locate the image you will add alt-text to.

    1. Images are marked by the "img" tag.

  3. Following the "" after the image path, type in your alt text using the following format.
