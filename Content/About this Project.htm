About this Project

I used a number of programs to complete this project, chiefly Madcap Flare to build the main website and PDF. I needed to navigate through the various files on Flare whilst focusing on the visual hierarchy of each page to ensure it was as visually pleasing as I could realistically make it. I utilized Flare's text editor feature to edit the code of my website if I could not create what I was envisioning with the standard XML editor.

 

I also utilized Adobe Illustrator to create my website logo, in which I focused on simplicity and personality to ensure I had an effective logo. For this I had to use a wide range of tools to properly understand how to create a proper logo, and I also had to understand how to create patterns as I was expected to add a pattern to the PDF. I learned how to make repeating patterns and how to adjust them, but mainly relied on the pen and curvature tools to create my logo.

 

Finally, I used Adobe Acrobat to check over my PDF and to ensure it was accessible. I used the program's accessibility checker to scan the document, and it listed out any areas that may need attention. It highlighted things such as alt text, indicating that I should check them manually, however there were no auto-identified errors in the document.
